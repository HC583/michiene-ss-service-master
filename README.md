## スマホアプリ風に使う方法

このプロジェクトはPWA対応済みです。スマホでURLを開いて、ブラウザのメニューから「ホーム画面に追加」を選ぶと、アプリのように起動できます。

アプリを開く時の合言葉:

```text
583
```

iPhoneの場合は Safari で開いて、共有ボタンから「ホーム画面に追加」を選びます。

Androidの場合は Chrome で開いて、メニューから「アプリをインストール」または「ホーム画面に追加」を選びます。

注意: スマホで本物のアプリのようにインストールするには、基本的にHTTPSのURLが必要です。自分のパソコンだけで試す場合は、同じWi-FiのURLで表示確認できます。
